Documentação do Projeto – Colmeia App
1. Nome do Projeto
  Colmeia App

2. Descrição do Projeto
  O Colmeia App é um site criado para ajudar trabalhadores autônomos a divulgar e vender seus produtos ou serviços por meio de uma vitrine virtual.
Cada vendedor possui uma página própria, onde o consumidor pode visualizar os produtos/serviços e, ao clicar em um botão, ser direcionado diretamente para o contato do vendedor (ex: WhatsApp ou outro meio).

3. Objetivo do Projeto
  Facilitar a divulgação de produtos e serviços de trabalhadores autônomos;
  Criar uma vitrine virtual simples e acessível;
  Conectar consumidores diretamente aos vendedores, sem intermediários;
  Oferecer uma solução prática e funcional através de um site.
  
4. Funcionalidades Implementadas (MVP)
  Até o momento, o projeto conta com as seguintes funcionalidades:
  Página principal do site;
  Estrutura de vitrine virtual para os vendedores;
  Página individual do vendedor;
  Botão de contato direto com o vendedor;
  Interface visual simples e intuitiva;
  Front-end totalmente funcional (HTML, CSS e Java).

5. Tecnologias Utilizadas

  HTML – Estrutura do site;
  CSS – Estilização e design das páginas;
  Java – Funcionalidades e lógica do site;
  GitHub – Versionamento e hospedagem do código.
